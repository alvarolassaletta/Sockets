 TCP Interactivo: “Chat simple 1 turno”

Objetivo: cliente manda mensaje,
servidor responde con otro mensaje (de teclado del servidor).

    Checklist

     Cliente lee del teclado y envía al servidor

     Servidor lee mensaje del cliente y lo imprime

     Servidor pide texto por teclado y responde

     Cliente imprime respuesta

     Control de errores y cierre
